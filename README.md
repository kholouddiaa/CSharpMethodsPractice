# C# Basics – Methods Practice

This repository contains C# console application examples focused on practicing
core programming fundamentals before learning Object-Oriented Programming (OOP).

The project emphasizes clean code, reusable methods, and safe user input handling.

---

## Topics Covered
- Console Input / Output
- Methods and Parameters
- Return Values
- Boolean Methods
- Input Validation using TryParse
- DRY Principle (Don’t Repeat Yourself)
- Single Responsibility Principle
- Basic Math Operations
- Even / Odd Check
